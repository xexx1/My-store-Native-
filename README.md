# Khan Store - แอปมือถือ React Native

ยินดีต้อนรับสู่ **Khan Store** ร้านธรรมดา ที่ยังไม่มีอะไร 

## ฟีเจอร์
- **แสดงสินค้า**: จัดเรียงสินค้ารองรับ Responsive

## ความต้องการเบื้องต้น

ก่อนรันแอปนี้ ต้องติดตั้งเครื่องมือต่อไปนี้:

- **Node.js**: แนะนำเวอร์ชันล่าสุด
- **Expo CLI**: สำหรับพัฒนาและทดสอบแอป React Native
- **NPM**: แนะนำสำหรับจัดการ dependencies
